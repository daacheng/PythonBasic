## 比较含退格的字符串
给定 S 和 T 两个字符串，当它们分别被输入到空白的文本编辑器后，判断二者是否相等，并返回结果。 # 代表退格字符。

注意：如果对空文本输入退格字符，文本继续为空。

示例：
```python
输入：S = "ab#c", T = "ad#c"
输出：true
解释：S 和 T 都会变成 “ac”。
```

代码：
* 碰到“#”就弹栈
* 碰到其他字符就入栈

```python
def backspaceCompare(S, T):
    s_stack = []
    t_stack = []
    for item in S:
        if item == '#' and s_stack:
            s_stack.pop()
        if item != '#':
            s_stack.append(item)

    for item in T:
        if item == '#' and t_stack:
            t_stack.pop()
        if item != '#':
            t_stack.append(item)

    return s_stack == t_stack
```
