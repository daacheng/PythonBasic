## 集合元素删除
#### remove(),删除元素，元素不存在会报错
```python
a=set('abcdefghijk')  
a.remove('a')  
a  # {'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i', 'j', 'k'}  
```

#### discard()删除元素，元素不存在不会报错
```python
a=set('abcdefghijk')  
a.discard('m')  
a  # {'a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i', 'j', 'k'}  
```
