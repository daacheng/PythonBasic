## 合并两个有序链表
将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

示例：
```python
输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4
```

### 代码
#### 递归
* 如果两个链表有一个是空链表，则返回另一个链表
* 如果两个链表都不是空：
```python
# 两个链表头部值较小的一个节点与剩下元素的 merge 操作结果合并。
l1[0]<=l2[0]     l1 = l1[0] + mergeTwoLists(l1[1:], l2)
l1[0]>l2[0]      l2 = l2[0] + mergeTwoLists(l1, l2[1:])
```

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

    def __str__(self):
        var_list = []
        var_list.append(self.val)
        while self.next:
            var_list.append(self.next.val)
            self = self.next
        return '->'.join([str(item) for item in var_list])

def mergeTwoLists(l1, l2):
    if l1 is None:
        return l2
    elif l2 is None:
        return l1
    elif l1.val <= l2.val:
        l1.next = mergeTwoLists(l1.next, l2)
        return l1
    elif l1.val > l2.val:
        l2.next = mergeTwoLists(l1, l2.next)
        return l2

if __name__ == '__main__':
    l1 = ListNode(1)
    l1.next = ListNode(2)
    l1.next.next = ListNode(4)

    l2 = ListNode(1)
    l2.next = ListNode(3)
    l2.next.next = ListNode(4)
    print(l1)
    print(l2)
    res = mergeTwoLists(l1, l2)
    print(res)
```

运行结果
```python
1->2->4
1->3->4
1->1->2->3->4->4
```

#### 迭代
* 哨兵节点prehead
* 指针prev

```python
def mergeTwoLists(l1, l2):
    prehead = ListNode(-1)

    prev = prehead
    while l1 and l2:
        if l1.val < l2.val:
            prev.next = l1
            l1 = l1.next
        else:
            prev.next = l2
            l2 = l2.next
        prev = prev.next

    if l1:
        prev.next = l1
    else:
        prev.next = l2
    return prehead.next
```
