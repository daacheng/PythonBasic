## 回文链表
请判断一个链表是否为回文链表。
#### 示例
```python
输入: 1->2->2->1
输出: true
输入: 1->2
输出: false
```

#### 代码
把链表转成列表，通过列表判断是否回文
```python
def isPalindrome(head):
    val_list = []
    cur = head
    while cur:
        val_list.append(cur.val)
        cur = cur.next
    return val_list == val_list[::-1]
```
